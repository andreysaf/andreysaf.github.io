DOCX Editor

Context

DOCX editor allows you to create and edit DOCX documents in a browser without any server-side dependencies or Microsoft Office licenses.

The editor is a zero-to-one product that I have built together with 2 product designers, 2 scrum teams, one frontend and one backend.

Use Cases

  • Edit DOCX
  • Create DOCX
  • Save as PDF
  • Save as DOCX
  • Create templates
